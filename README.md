# ระบบรายรับรายจ่าย

เว็บแอปสำหรับจัดการรายรับและรายจ่าย พร้อมหมวดหมู่ และรายงานประจำวัน

## คุณสมบัติ
- ล็อกอินด้วยรหัสเดียว (รหัสเริ่มต้น `1234`)
- เพิ่มรายการรายรับหรือรายจ่ายได้
- แยกหมวดหมู่และกำหนดหมวดหมู่เองได้
- เลือกธนาคารสำหรับแต่ละรายการ และดูสรุปตามธนาคาร
- ดูรายงานวันนี้และสรุป 7 วันล่าสุด
- ล้างข้อมูลทั้งหมดได้

## การใช้งาน
1. เปิดไฟล์ `index.html` ในเบราว์เซอร์
2. ล็อกอินด้วยรหัส `1234` หรือรหัสที่ตั้งเอง
3. เพิ่มรายการ และดูรายงานทันที

> ข้อมูลจะถูกเก็บไว้ในเบราว์เซอร์ด้วย `localStorage`
